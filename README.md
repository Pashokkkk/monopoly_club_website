# 🌐 Monopoly Club website

A multi-page website created using **HTML**, **CSS**, and **JavaScript**.

This project includes several pages:
- 🏠 Home (index.html)
- 🎮 Game (game.html)
- 📝 About Us (about_us.html)
- 📞 Contact (contact.html)
- 📅 Events (events.html)
- 🔑 Sign Up (sign_up.html)
